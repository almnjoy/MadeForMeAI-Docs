Upload a file into the chat and ask MindMap to work with it. Since MindMap runs in the cloud and can't see your computer, uploading is how you hand it a file. [Why MindMap is in the cloud](/understanding/cloud-not-local)

## What you can upload

- **Documents** PDF, Word (.docx), text files
- **Spreadsheets** CSV, Excel
- **Images** PNG, JPG, screenshots
- **Code files** any plain text file

## How to upload

Click the attachment icon in the chat input, pick your file, and send it with your message.

{/* MEDIA gif: clicking the attachment icon, selecting a PDF, sending it with a question | caption: "Attach a file and ask in the same message." */}

## What to ask it to do

Once the file is in, tell MindMap what you need:

- _"Summarize this contract in plain English."_
- _"What are the key numbers in this spreadsheet?"_
- _"Find any errors in this document."_
- _"Rewrite this in a friendlier tone."_
- _"What's in this screenshot?"_

Reading the file you uploaded is a read, so it happens instantly. If you ask MindMap to then write the result somewhere (save a file, send an email), that part queues for your approval. [Tool Calls](/understanding/tool-calls)

## Tips

- Attach several files in one message
- A larger file takes a few seconds to process, so give it a moment
- For a long document, point to the part you care about. _"Focus on section 3"_ beats _"read this."_
