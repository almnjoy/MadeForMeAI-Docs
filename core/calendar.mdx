A simple calendar the agent can read and write for you. Ask what is coming up and it lists your
upcoming events. Tell it to add something and it puts an event on a date and time, with an optional
location. Reading your events runs immediately. Adding an event is a write, so the agent queues it
and it only goes on the calendar after you approve. This is the local, built-in calendar that lives
on your box. Your real Outlook or Microsoft 365 calendar is a separate connection (the n8n
Automations feature), not this one.

## HOW TO SETUP

Nothing to set up. Calendar is on by default and needs no key. Open MindMap and ask the agent
about your schedule and it works.

If you ever want to confirm it is on, open MindMap -> Admin (marketplace) -> the Calendar card.
It is enabled out of the box.

## WHAT USER NEEDS TO GET

Nothing.

## HOW IT WORKS

- The agent lists your upcoming events when you ask. This is a read, so it runs immediately.
- Adding an event is a write. The agent queues a proposal with the date, time, title, and any
  location you gave, and it only fires after you approve. The agent never adds anything on its own.
- Your events are stored locally on your box. They are not sent anywhere and not shown in the code.
- There is no key for this feature, so there is nothing to rotate or paste.

## WHAT CAN BE CALLED VIA AI

- "what's on my calendar" (read, runs immediately)
- "what do I have coming up this week" (read, runs immediately)
- "add a meeting Thursday at 3pm called Vendor call" (write, queued for approval)
- "put a dentist appointment on Friday morning" (write, queued for approval)
- "add an event Monday at 10am called Site visit at 400 Main St" (write, queued for approval)

The agent reads your events on its own and queues every new event for you to approve.

## LIMITATIONS

- It is a local list. It does not sync to Outlook or Google. For Microsoft 365, use the n8n
  Automations feature instead.
- It does not send calendar invites to anyone.
- It does not send reminders or notifications before an event.
- It tracks single events with a time and an optional location. It is not a full scheduling tool
  with attendees, recurring series, or availability checks.

## FAQ

**Does this update my real Outlook or Google calendar?** No. This is a separate local calendar on
your box. Your Outlook or Microsoft 365 calendar connects through the n8n Automations feature, which
is its own setup.

**Will it remind me before a meeting?** No. It stores the event so you and the agent can see it, but
it does not send reminders or push notifications.

**Can the agent add an event without asking me?** No. Reading your events runs on its own, but every
new event queues for your approval and only goes on the calendar after you approve it.

**Do I need to turn it on or add a key?** No. Calendar is on by default with no key. Just ask the
agent about your schedule.
