Ask for a quick visual answer and the agent draws it as a tile instead of a paragraph. You get current weather with a 5-day forecast, a live stock, ETF, or crypto quote with an intraday sparkline, a company logo, a real photo, a map of a place, and data charts as a bar, line, or pie. These are reads, so they run immediately. Nothing here changes anything on your account, so there is no approval step.

## HOW TO SETUP

Nothing to do. Weather, stocks, logos, photos, and charts are on by default and need no key. Open MindMap and ask.

<Note>Maps that need an API key are not user-configurable in the UI today. There is no in-app field for a Google Maps key. Weather, stocks, and logos work with no key.</Note>

## WHAT USER NEEDS TO GET

Nothing. Weather, stocks, logos, photos, and charts all work out of the box.

## HOW IT WORKS

- You ask in plain language. The agent picks the right widget and renders a tile in your reply: a weather card, a quote with a sparkline, a logo, a photo, a map, or a chart.
- Weather comes from Open-Meteo, quotes from Yahoo Finance, and logos from Clearbit. These are free public sources, so the agent reads them live each time you ask.
- Charts are drawn by a renderer built into MindMap. The agent charts numbers you give it or numbers it just looked up. The chart is a snapshot of those numbers, not a feed that keeps updating.
- Maps that need an API key are not set up in the UI today. Weather, stocks, logos, photos, and charts still work.
- Every widget is a read, so it runs on its own with no approval prompt. None of them send, post, or change anything.

## WHAT CAN BE CALLED VIA AI

All of these are reads and run immediately:

- "weather in Lansing": weather card with a 5-day forecast (read)
- "show AAPL": live quote with an intraday sparkline (read)
- "how is the S&P doing": index quote and sparkline (read)
- "show the Stripe logo": company logo tile (read)
- "show a picture of the Chicago skyline": photo tile (read)
- "chart these monthly sales as a bar graph": chart tile from the numbers you give it (read)

## LIMITATIONS

- Weather, stocks, and logos use free public sources, so expect the occasional gap, stale value, or short delay.
- Maps that need an API key are not configurable in the UI today.
- Charts visualize numbers you hand the agent or numbers it looked up for you. A chart is not a live data feed and will not refresh on its own.
- A logo lookup depends on the company being in Clearbit's set. A small or new company may not return one.
- Stock quotes are for quick reference and can lag the real market. Do not trade off them.

## FAQ

**Do I need to set anything up to use widgets?** No. Weather, stocks, logos, photos, and charts are on by default with no key.

**Can I show maps?** Maps that need an API key are not configurable in the UI today. There is no in-app field for a Google Maps key.

**Are the stock quotes real-time?** They are live reads from a free public source and meant for a quick glance. They can run a little behind the real market, so treat them as a reference and not a trading signal.

**Can a chart pull my live numbers automatically?** No. A chart draws the numbers you give the agent in your message or numbers it looked up while answering you. It is a snapshot, so ask again to redraw it with fresh numbers.
