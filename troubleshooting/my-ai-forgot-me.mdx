---
title: "MindMap Forgot Me"
description: "Why MindMap can seem to forget you, how memory works per tenant, and how to save a fact that sticks."
---

If MindMap acts like it's meeting you for the first time, its long-term memory probably wasn't wiped. Usually you're just looking at an empty chat, or you're on a different surface. Here's how to tell the difference and get back on track.

## How MindMap memory works

Memory in MindMap is **per tenant**, shared across your whole team's workspace. It is not tied to one chat thread. Two things follow from that:

- Saved memories persist even after a conversation is gone.
- An empty-looking chat does not mean lost memory. It means that thread has no history yet.

## You cleared the chat

Clearing a chat (or starting a new one) wipes the visible conversation, not the saved memories behind it. The thread starts blank, so MindMap won't recap what you just discussed, but anything you told it to remember is still there.

Start typing and ask about a saved fact to confirm. If it knows, memory is intact.

## You're on a different surface or Space

MindMap has more than one place to talk to it: the canvas, the chat app, the pop-up widget, Telegram, and any Spaces your team set up. Each surface keeps its own conversation history, and a Space can run with its own persona and tools.

So if a chat looks empty, check that you're on the surface (or in the Space) where you were working. Switching back brings that conversation history with you. Shared tenant memory is available from every surface either way.

## Save a memory that sticks

To store something durable, tell MindMap plainly and ask it to remember:

> Remember that I'm the ops lead, I prefer short replies, and our fiscal year starts in April.

MindMap saves what you ask it to keep, and pulls it back on later messages from any surface. To top up after a blank thread, send one message with the essentials:

```
Remember these about me:
- My name and role: [name, role]
- What I'm focused on right now: [current priorities]
- How I like replies: [tone, length, format]
- Key people or projects: [names and context]
```

<Note>
You can review and search what's been saved on the Memory page. If a fact is wrong or stale, tell MindMap to update it, the same way you saved it.
</Note>
