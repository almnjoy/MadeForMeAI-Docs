MindMap is one AI workspace where your agent can use every tool you connect: your CRM, email, calendar, code repos, the web, and more. You work two ways. **Canvas** lays answers out as tiles you can drag, arrange, and save into dashboards. **Chat** is a plain linear thread, the default on phones. Same agent, same keys, same memory behind both.

Every tool is a node you can see or a toggle you flip. Reads run the moment you ask. Every write queues for your approval and only fires after you click approve.

## Approval-first by design

The agent reads on its own: it lists, searches, fetches, and looks things up the moment you ask. Every write is different. Send an email, post a message, create an issue, change a record, and the agent queues the action and shows it to you first. Nothing sends or changes until you approve it. If there is one write you repeat and trust, a Trust Rule pre-approves that narrow case so you stop clicking for it.
