# Voice Calls (Retell)

Outbound AI phone calls. A demo call to your own number, and approval-gated follow-up calls to
contacts behind a TCPA voice-consent gate. Call summary, sentiment, and transcript stream back from
Retell webhooks. Inbound capture stays in your existing n8n RETELL flows.

## HOW TO SETUP
1. Create a Retell account. Get the API key, buy/assign a phone number, create ONE outbound agent.
2. Marketplace -> Voice Calls. Paste `RETELL_API_KEY`, `RETELL_FROM_NUMBER` (E.164, e.g. +1303...),
   `RETELL_AGENT_ID`.
3. In the Retell dashboard set the call webhook to `https://your-tenant.madeformeai.com/api/webhooks/retell`.
4. First-run only: set `RETELL_WEBHOOK_INSECURE=true` in the api .env until you confirm Retell's
   signature scheme, then remove it so signatures are enforced.

## WHAT USER NEEDS TO GET
A Retell account, an API key, a phone number, and one outbound agent id.

## HOW IT WORKS
`demo_call`/`follow_up_call` queue an approval. On approve, the action calls Retell create-phone-call
with dynamic variables (person + topic) and metadata (intent + CRM id). Retell webhooks
(call_started/ended/analyzed) hit /api/webhooks/retell and update the voice_calls table. `call_status`
reads the result.

## WHAT CAN BE CALLED VIA AI
- "Demo call my phone +1..." -> demo_call (no third-party consent needed)
- "Call Sarah about her renewal, she consented" -> follow_up_call (needs voice_consent=true)
- "How did call <id> go?" -> call_status
- "List my Retell agents and numbers" -> list_voice_agents

## LIMITATIONS
- Outbound only here; inbound is your n8n flows.
- follow_up_call is REFUSED on approve unless voice_consent is true (TCPA). Honor DNC + 8am-9pm
  callee-local + recording disclosure in two-party states.
- Confirm Retell's webhook signature scheme before trusting it in production.

## FAQ
Q: Will it dial without me? A: No. Every call is approval-gated; nothing dials until you approve.
Q: TCPA risk on the demo? A: None, the demo calls your own consented number.
Q: Standard vs fun agent? A: demo_call uses the standard MindMap Demo agent by default; pass fun=true (or say "the fun one") for the Wandering AI RPG agent.
