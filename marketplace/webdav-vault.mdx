Stores your notes as plain `.md` files in a WebDAV vault, two-way synced with Obsidian. It is the
storage behind the Notes tool and the Memory graph. What you write in MindMap shows up in Obsidian
and what you write in Obsidian shows up in MindMap.

## SETUP

Your vault is set up for you. The address and login are provisioned and connected before you log
in, so there are no keys to enter. When it is on, the Notes tool and Memory graph read and write
the vault. When off, notes fall back to a local table.

For Obsidian two-way sync, we give you the vault address, username, and password. Install the
**Remotely Save** plugin in Obsidian, set it to WebDAV with those values, and sync.

## HOW IT WORKS

- Notes are `.md` files in the vault root. The note title is the filename, so renaming the title
  renames the file.
- The vault speaks WebDAV over HTTPS, so any WebDAV client (like Obsidian) can sync the same files.
- Your vault is private to you: a separate address, login, and folder, so there is no leak between
  users.

## WHAT CAN BE CALLED VIA AI

Through the Notes tool: "save a note titled X", "read my Y note", "append this to Z", "list my
notes". The Memory graph shows how they link.

## LIMITATIONS

- Top-level files only for now (subfolders not yet recursed).
- The vault uses its own username and password, not your main sign-in. That is what lets Obsidian
  connect.
- Last-writer-wins on a clash (no merge). Editing the same file in Obsidian and MindMap at the
  same instant can overwrite one side.

## FAQ

**Is this Obsidian?** No. It is a WebDAV server over your `.md` files. Obsidian (or any WebDAV
client) syncs to the same files.

**Where are my files?** In your own private vault folder on the MadeForMeAI cloud.

**What if I turn it off?** Notes fall back to a local database table; the vault files stay put.
