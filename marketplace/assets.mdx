Object storage for files your agents generate: proposals, exports, images, PDFs. Browse, preview,
download, upload, delete.

## SETUP

Assets is set up for you. Your storage is already provisioned and connected before you log in,
so there are no keys or endpoints to enter. When it is on, the Assets page and nav appear, and the
agent can `save_asset` and `list_assets`.

## HOW IT WORKS

- The Assets page lists objects with size, source (agent or upload), and date. Click a file to
  preview (images render inline) or download.
- Downloads use a short-lived presigned URL straight from storage, so big files do not stream
  through the app.
- Upload via the page (pick a file) or have the agent `save_asset` a generated document. Each file
  carries provenance metadata (which agent/skill made it).

## WHAT CAN BE CALLED VIA AI

- "save this as an asset" / "export this proposal to a file" -> `save_asset`
- "what files do I have in Assets" -> `list_assets`

## LIMITATIONS

- Web upload is a single file up to 25 MB (no multipart or folder upload yet).
- Top-level keys; no folder browser yet.
- The agent saves text/markdown content; binary outputs (images, PDFs) come from whatever feature
  generates them, then land here.

## FAQ

**Where are the files?** In your own private storage, set up for you on the MadeForMeAI cloud.

**Is my storage private?** Yes. Downloads use short-lived links, and you can see only your own
files.
