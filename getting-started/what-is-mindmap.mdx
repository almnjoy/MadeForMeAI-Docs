MindMap is your business on one canvas. It is a white-label AI workspace where every tool the agent touches becomes a node you can see and click into. Email, billing, CRM, network status, and thousands of apps through one connector, all in one place.

You talk to one agent. It works across everything you have wired up, and it shows its work instead of hiding behind a chat box.

## One agent, two surfaces

Chat and Canvas are not two different agents. They are two ways to talk to the one assistant. Same agent, same keys, same memory behind both.

<CardGroup cols={2}>
  <Card title="Canvas" icon="diagram-project">
    A visual board where the apps and data the agent works with show up as tiles you can arrange, save, and click into.
  </Card>
  <Card title="Chat" icon="comments">
    A full chat workspace where you type and the reply streams in, with results landing as inline cards. Default on phones.
  </Card>
</CardGroup>

Switch between them anytime. What you do in one shows up in the other, because it is the same agent with one memory and one approval gate. A write you approve in Chat is the same kind of action you would approve on the Canvas.

## Every tool is a node you can toggle

Nothing is hidden. Each thing the agent can do is a node on the Canvas or a card in the marketplace. You see what is connected, you turn pieces on and off, and you click a node to open what is behind it.

## One connector reaches many apps

Wire up a connector like [Zapier](/marketplace/zapier) or [Composio](/marketplace/composio) once, and the agent can use everything those connectors reach. You set up the connection in Admin, and the apps behind it become available without more plumbing.

## The marketplace and its four buckets

Everything the agent can do comes from the marketplace in Admin. It is organized into four buckets:

<CardGroup cols={2}>
  <Card title="Apps" icon="grid">
    Full surfaces like Money, Sales, Morning Brief, and the Chat workspace.
  </Card>
  <Card title="Tools" icon="screwdriver-wrench">
    Connectors and integrations the agent calls: CRM, email, voice, browser, storage, network.
  </Card>
  <Card title="Skills" icon="wand-magic-sparkles">
    Multi-step flows like Deep Research and Lead Outreach that the agent runs end to end.
  </Card>
  <Card title="Secrets" icon="key">
    API keys and tokens, saved write-only to your box, never shown back to the browser.
  </Card>
</CardGroup>

You flip a card on, and it applies on your next message. No restart.

## Approval-first by design

MindMap never sends or changes anything on its own.

- **Reads run live.** Checking a balance, reading your pipeline, pulling network status. Instant.
- **Writes wait for you.** Sending an email, creating a lead, moving money. Queued for your approval.
- **Trust Rules let you grant once.** When you are ready, approve an action type so it runs by itself next time. Secrets can never be auto-approved.

<Note>
**Cloud hosted:** you'll be provided a secure login via email. Sign-in is handled for you behind the scenes. **Or self hosted (coming soon):** bring your own authentication, enabled in configuration.
</Note>
